# VuePress-Blog
Powered by [vuepress-theme-reco](https://github.com/vuepress-reco/vuepress-theme-reco)


## Environment
* Yarn 1.22.17


## Getting Started
Install dependencies:
```shell
yarn install
```

## Build
Build directory as a static site:
```shell
yarn build
```

## Run
Serve the documentation site in the local server:
```shell
yarn dev
```
